本项目可以:

1.  检查本地是否有 WormHole；

2.  检查本机 package list，是否可能存有百度系的APP；

3.  package-list.txt 为百度系的 Android APP 的 package 列表，欢迎补充；

4.  clean.sh 使用 adb uninstall 命令卸载 package-list.txt 中的所有 APP。
